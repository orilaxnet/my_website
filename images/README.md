# Images Folder

This folder contains all project images for the portfolio.

## Usage

When using the Projects Editor (`projectseditor.html`), uploaded images should be copied to this folder.

The editor will generate filenames and paths like:
- `images/1234567890_0_photo.jpg`
- `images/1234567890_1_image.jpg`

## Instructions

1. Upload images using the Projects Editor
2. Click "Download Images" button
3. Copy the downloaded images to this folder
4. Make sure the filenames match exactly as shown in the editor
